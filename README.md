# Grip_React_JeongSohyun

### 🗨 소개
**영화 검색과 즐겨찾기가 가능한 웹사이트입니다.**  
화면은 모바일형으로 제작했습니다.  
사전과제를 위해 React를 처음 다뤄봤습니다.

### 🎁사용한 오픈소스
#### sweetalert2 
즐겨찾기 추가 및 취소 시 alert 창

#### react-js-pagination

### ➕콘솔에 추가해야할 것들
